# NovaStack AI Landing Page

A premium, responsive frontend website for a modern AI and SaaS startup. This project was built with semantic HTML5, CSS3, and vanilla JavaScript only, making it a strong fit for an internship submission and GitHub portfolio.

## Project Overview

NovaStack AI showcases a polished product studio experience with a modern glassmorphism aesthetic, premium gradients, smooth motion, and a fully responsive layout. The page is designed to feel like a real startup landing page with strong visual hierarchy and modern interaction patterns.

## Features

- Sticky navigation with a mobile-friendly menu
- Hero section with clear primary and secondary CTAs
- About, features, services, portfolio, and team sections
- Animated statistics counters
- Testimonials slider with dots and navigation controls
- FAQ accordion for content organization
- Contact form with client-side validation and success feedback
- Theme toggle, smooth scrolling, reveal animations, and back-to-top button
- Semantic HTML and enhanced accessibility support

## Folder Structure

- index.html — Page structure and content
- style.css — Responsive styling, glassmorphism visuals, animations, and layout
- script.js — Navigation, reveal effects, counters, FAQ, slider, form validation, and theme switching
- assets/ — Local SVG illustrations, favicon, and social preview assets

## Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript

## How to Run

1. Open the project folder in your browser directly, or serve it locally.
2. If you prefer a local server, run:
   - Python: `python -m http.server 8000`
3. Then visit `http://localhost:8000`.
